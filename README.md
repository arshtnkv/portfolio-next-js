# Portfolio Next.js

URL: [arshtnkv.ru](https://arshtnkv.ru/)

![Cover Image](public/cover.png)

### Технологии:

- Next.js
- TypeScript
- TailwindCSS
- ESLint
- Prettier
- Framer Motion
- AOS

### Основные команды для работы:

- Установка

```
npm i
```

- Запуск локального сервера

```
npm run dev
```

- Запуск ESLint

```
npm run lint
```
